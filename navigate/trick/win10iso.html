Windows 10 ISO và Rufus
Đây là nơi luu file Windows 10 22H2 ISO cùng với phần mềm Rufus 4.9 bản portable để các bạn có thể cài đặt và sử dụng ngay và luôn

Yêu cầu

- USB dung lượng từ 6gb trở lên (tốt nhất là trên 8gb)
- Máy tính (điều đương nhiên)
- Các file có link tải bên dưới

Danh sách file

Click vào tên file để tải
Tên
1 Windows 10 ISO
2 Rufus

Hướng dẫn sử dụng

- Mở rufus, tiến hành cắm usb vào máy tính
- Tại mục đầu tiên (device) hãy chọn vào danh sách sổ xuống và chọn đúng vào tên usb
- Tại mục thứ 2 (Boot selection) hãy nhấn nút select ở cuối dòng này sau đó chọn vào đúng file iso (file 1) đã tải
- Nhấn start ở bên dưới cùng phần mềm
- Chờ cho thanh màu xanh chạy xong và hiện complete thì tắt phần mềm rút usb
- Cắm vào thiết bị cần cài. Vào menu boot và chọn vào tên của usb
- Việc còn lại thì tùy chỉnh theo ý của bạn (hoặc có thể xem video trên youtube cài win từ lúc này)
